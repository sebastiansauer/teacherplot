# Uniform plot

Plots a uniform distribution with parameters a and b, using ggplot2

## Usage

``` r
uniform_plot(a, b)
```

## Arguments

- a:

  minimum value of the uniform distribution

- b:

  maximum value of the uniform distribution

## Value

ggplot2 object

## Examples

``` r
uniform_plot(0, 1)
```
