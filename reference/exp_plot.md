# Exponential distribution plot

Exponential distribution plot

## Usage

``` r
exp_plot(r, max_x = 25, probs = c(0.05, 0.25, 0.5, 0.75, 0.95))
```

## Arguments

- r:

  rate parameter

- max_x:

  max value x axis

- probs:

  quantiles to indicate in the plot

## Value

ggplot2 object

## Examples

``` r
exp_plot(0.1)
#> Error in loadNamespace(x): there is no package called ‘ggplot2’
```
